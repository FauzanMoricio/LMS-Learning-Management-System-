# Project Title
**LMS-Learning-Management-System**

## Deskripsi
Learning Management System (LMS) adalah sebuah platform perangkat lunak yang digunakan untuk mengelola, menyampaikan,mengupload dan memantau proses pembelajaran secara online pada SMK al-falah.

Dibuat Menggunakan:  
> HTML, CSS, PHP, MySql, JavaScript.
